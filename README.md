# 支払いスケジュール管理アプリ

React + Vite で構築された支払いスケジュール管理 Web アプリケーションです。

## 機能

- **ダッシュボード**: 未払い・期限超過・支払済の合計金額と件数を一覧表示
- **7日以内の支払い予定**: 直近の支払いをアラート表示
- **支払い一覧**: ステータス別フィルタリング（すべて・未払い・期限超過・支払済）
- **支払い追加・編集**: タイトル、金額、期日、カテゴリ、繰り返し設定、メモ
- **支払済にする**: ワンクリックでステータス更新
- **削除**: 確認ダイアログ付き削除
- **ローカルストレージ**: データをブラウザに永続保存

## カテゴリ

住居費 / 光熱費 / クレジット / 保険 / サブスク / 税金 / その他

## 繰り返し設定

一回払い / 毎週 / 毎月 / 毎年

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```
